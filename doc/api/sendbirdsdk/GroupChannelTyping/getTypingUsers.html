List<User> getTypingUsers ()

Returns a list of User who are currently typing.

Implementation

List<User> getTypingUsers() {
  final typingStatuses =
      _sdk.cache.findAll<TypingStatus>(channelKey: channelUrl);
  return typingStatuses.map((e) => e.user).toList();
}
